ProtonVPN Accessibility – NVDA Add-on

Purpose

Enhances ProtonVPN (Windows) accessibility with NVDA via UI Automation (XAML): control labeling and dynamic information reading.

Features

  • Button fixes: Unlabeled or poorly labeled buttons are automatically corrected
  • Dynamic announcements: Label + value announced together (IP, country, provider, etc.)
  • Connected VPN support: VPN IP, total traffic, real-time traffic
  • Promotional cards: Overlays and promo cards are properly labeled

Keyboard Shortcuts

The following shortcuts are available only when ProtonVPN is in the foreground:

Shortcut Action
Ctrl+Shift+D Connect / Disconnect VPN
Ctrl+Shift+K Toggle Kill Switch
Ctrl+Shift+C Open country selector
Ctrl+Shift+T Announce traffic info

Enhanced Announcements

Element Example
Local IP "Your IP address: 212.133.41.101"
Country "Country: France"
Provider "Provider: Lumen"
VPN IP "VPN IP address: 37.19.199.137"
Total traffic "Total traffic: 12 MB"
Current traffic "Current traffic: 416 B/s, 0 B/s"

Compatibility

  • NVDA 2023.1 and later
  • ProtonVPN for Windows
