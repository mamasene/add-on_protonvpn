ProtonVPN Accessibility – NVDA Add-on

Objectif

Améliore l'accessibilité de ProtonVPN (Windows) avec NVDA via UI Automation (XAML) : labellisation des contrôles et lecture d'informations dynamiques.

Fonctionnalités

  • Correction des boutons : Les boutons sans nom ou mal labellisés sont corrigés automatiquement
  • Annonces dynamiques : Label + valeur annoncés ensemble (IP, pays, fournisseur, etc.)
  • Support VPN connecté : IP VPN, trafic total, trafic en temps réel
  • Cartes promotionnelles : Les overlays et cartes promo sont labellisés correctement

Raccourcis clavier

Les raccourcis suivants sont disponibles uniquement lorsque ProtonVPN est au premier plan :

Raccourci Action
Ctrl+Shift+D Connecter / Déconnecter le VPN
Ctrl+Shift+K Activer / Désactiver le Kill Switch
Ctrl+Shift+C Ouvrir le sélecteur de pays
Ctrl+Shift+T Annoncer le trafic (actuel + total)

Annonces améliorées

Élément Exemple d'annonce
IP locale "Votre adresse IP : 212.133.41.101"
Pays "Pays : France"
Fournisseur "Fournisseur : Lumen"
IP VPN "Adresse IP du VPN : 37.19.199.137"
Trafic total "Trafic total : 12 Mo"
Trafic actuel "Trafic actuel : 416 o/s, 0 o/s"

Compatibilité

  • NVDA 2023.1 et versions ultérieures
  • ProtonVPN pour Windows
